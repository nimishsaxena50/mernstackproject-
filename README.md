# MERN Stack Project - Project Manager

## 🚀 Live Demo
https://mernstackproject-jrbr.vercel.app/

## 📌 Features
- User Authentication (Signup/Login)
- Create, Update, Delete Tasks
- Dashboard for managing projects
- Admin Invite System

## 🛠️ Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: MongoDB
- Deployment: Vercel (Frontend), Render (Backend)

## ⚙️ Installation (Local Setup)

### 1. Clone the repo
git clone https://github.com/nimishsaxena50/mernstackproject-.git

### 2. Install dependencies
cd frontend
npm install

cd ../backend
npm install

### 3. Run project
Frontend:
npm run dev

Backend:
npm run start
